## âš¡ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/interactive-canvas.git
    cd interactive-canvas
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment:**


    ```bash
    npm run dev
    ```
